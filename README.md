To-Do List

Este é um projeto simples de lista de tarefas feito com HTML, CSS e JavaScript.

A aplicação permite adicionar tarefas, definir uma prioridade e apagar tarefas.
As tarefas também ficam salvas no navegador usando localStorage, então elas continuam lá mesmo depois de atualizar a página.

Funcionalidades

Adicionar novas tarefas

Definir prioridade (Baixa, Média, Alta ou Urgente)

Exibir tarefas em uma lista

Apagar tarefas

Salvar tarefas no navegador

🛠 Tecnologias usadas

HTML

CSS

JavaScript

Objetivo

Este projeto foi feito para praticar:

manipulação do DOM

eventos em JavaScript

uso de arrays e objetos

armazenamento de dados com localStorage
